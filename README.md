#  Expense Tracker Web App

A modern and responsive **Expense Tracker Web Application** built using **HTML, CSS, and JavaScript**.  
This app helps users manage their income and expenses efficiently with real-time balance updates and local data storage.

---

## 📌 Features

### ✅ Core Features
- ➕ Add income and expense transactions  
- ❌ Delete transactions  
- 💰 Real-time balance calculation  
- 📊 Income & Expense summary  
- 💾 Data persistence using LocalStorage  
- 📱 Fully responsive design  

### 🌟 Advanced Features (Optional Enhancements)
- 📊 Expense analytics (charts)
- 🔍 Filter transactions (type/category/date)
- 🌙 Dark mode toggle
- ⚡ Smooth animations

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3 (Flexbox & Grid)** – Styling & Layout  
- **JavaScript** – Logic & Functionality  
- **LocalStorage API** – Data persistence
